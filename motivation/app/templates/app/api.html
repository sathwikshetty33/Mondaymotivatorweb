Getting Started

Our API is completely free to use and requires no authentication. Get inspired with our curated motivational quotes!

Get Quote

GET
https://sathwik33.pythonanywhere.com/api/
Description

Returns a motivational quote. No parameters required.

Example Response
{ "status": "success", "quote": "When you arise in the morning think of what a privilege it is to be alive, to think, to enjoy, to love...", "author": "Marcus Aurelius" }
Response Format
  • status (string): The status of the request
  • quote (string): The motivational quote
  • author (string): The author of the quote

Rate Limits

To ensure fair usage, we apply the following rate limits:

  • 100 requests per hour per IP address
  • 1000 requests per day per IP address

Error Handling

The API uses standard HTTP response codes:

  • 200: Success
  • 429: Too many requests - you've hit the rate limit
  • 500: Server error
