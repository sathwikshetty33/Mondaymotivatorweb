Get Weekly Quote

GET
https://sathwik33.pythonanywhere.com/api/
Description

Returns this week's motivational quote. No parameters required.

Example Response
{ "quote": "The only way to do great work is to love what you do.", "author": "Steve Jobs", "week_of": "2024-11-17", "next_update": "2024-11-24" }
Response Format
  • quote (string): The weekly motivational quote
  • author (string): The author of the quote
  • week_of (string): The start date of the current week
  • next_update (string): The date when the next quote will be available

Rate Limits

To ensure fair usage, we apply the following rate limits:

  • 100 requests per hour per IP address
  • 1000 requests per day per IP address

Error Handling

The API uses standard HTTP response codes:

  • 200: Success
  • 429: Too many requests - you've hit the rate limit
  • 500: Server error
